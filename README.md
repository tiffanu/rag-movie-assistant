# RAG Movie Assistant
**Глубокий семантический помощник по поиску фильмов**

Внутренний инструмент для подбора кино по сюжету, настроению, эпохе, стилю, актёрам, визуалу или любому сложному описанию на естественном языке.  
Работает на RAG-архитектуре с локальными моделями.

### Технологии
- **Эмбеддинги**: `sentence-transformers/all-MiniLM-L6-v2`
- **LLM**: `mistralai/Mistral-7B-Instruct-v0.2`
- **Векторная БД**: Chroma (локальный персистентный индекс)
- **Интерфейс**: Streamlit
- **Python**: 3.10+

### Возможности
- Двухэтапный поиск: широкий ретривал → точный отбор LLM
- Чёткий нумерованный список рекомендаций с годами выпуска
- Полностью оффлайн-режим после первой загрузки моделей
- Все ключи и секреты через `.env`


### Быстрый старт

1. Клонируем репозиторий
   ```bash
   git clone <ваш-репозиторий>
   cd rag-movie-assistant
   
2. Устанавливаем зависимости
   ```bash
   pip install -r requirements.txt
   
3. Вставляем Mistral API Key в .env.example
   ```
   MISTRAL_API_KEY=paste_your_mistral_api_key_here

4. Добавляем API ключ в окружение
   ```bash
   cp .env.example .env

5. Запуск
   ```bash
   streamlit run app_streamlit.py

Первый запуск скачает модели и загрузит базу (~1–2 минуты), последующие — мгновенно

### Примеры запросов (на английском)

- Фильмы вроде «Интерстеллара», но больше философии и без хэппи-энда
- Атмосферный киберпанк-нуар в дождливом городе, как «Бегущий по лезвию 2049»
- Корейские триллеры про месть круче «Олдбоя»

### Безопасность

API-ключи загружаются исключительно из .env (никогда не хардкодятся и игнорируются git).
